Can you explain what a decorator does in Python?
You · 2m ago
A decorator wraps a function to modify or extend its behaviour — without changing the original code.

Think of it like a gift box: @my_decorator placed above a function automatically runs the decorator logic every time that function is called.

Common uses include logging, auth checks, caching, and performance timing.
